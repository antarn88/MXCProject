# Node.js / NestJS backend projekt

## Függőségek telepítése

`yarn` parancs futtatása.

Szükséges a `.env` fájl másolása a projekt gyökerébe!

## Projekt indítása

`yarn start` parancs futtatása.

## Swagger

A swagger dokumentáció elérhető a következő címen:

`http://localhost:5000`

## Teszt user

Felhasználónév:
`tesztelek`
